Android App / PWA assets

Esta carpeta contiene los archivos de la PWA usados por la aplicación web.

Para ver la aplicación completa abre la URL principal:

http://192.168.101.10:8000/

Si quieres acceder al manifiesto o al service worker, usa estos archivos directamente:

  • manifest.json
  • pwa.js
  • service-worker.js

El directorio /android-app/ ya no mostrará un error Forbidden.
